# Ecommerce Sales Dashboard

## Overview
This project is an interactive Ecommerce Sales Dashboard built using Microsoft Power BI to analyze sales performance, customer behavior, profit trends, and business insights.

The dashboard helps businesses track important KPIs such as:
- Total Sales
- Total Profit
- Total Quantity Sold
- Average Order Value (AOV)

## Objectives
- Analyze ecommerce sales performance
- Identify top-performing states and categories
- Monitor monthly profit trends
- Understand customer purchasing behavior
- Track preferred payment methods

---

## Key Insights
- Maharashtra generated the highest sales revenue
- Clothing category contributed the highest quantity sold
- COD was the most preferred payment mode
- November recorded the highest monthly profit
- Some months showed negative profit trends, indicating business loss periods
